Keyboard Controls:
            ────────────────────────────────────
            [W] / [S]     →  Throttle
            [A] / [D]     →  Turn (50% speed)
            [←] / [→]     →  Strafe
            [1] / [2]     →  Adjust Velocity
            [3] / [4]     →  Adjust Acceleration
            [5] / [6]     →  Adjust Deceleration
            [R]           →  Change Robot
            [F]           →  Change Field
            [H]           →  Toggle Telemetry

                      Joystick Controls:
            ────────────────────────────────────
            [Left Stick]   →  Throttle + Strafe
            [Right Stick]  →  Turn
