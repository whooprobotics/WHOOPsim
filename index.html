Keyboard Controls:
            ────────────────────────────────────
            [W] / [S]     →  Throttle
            [A] / [D]     →  Turn (50% speed)
            [Q] / [E]     →  Adjust Acceleration
            [F]           →  Change Field
            [H]           →  Toggle Telemetry

                      Joystick Controls:
            ────────────────────────────────────
            [Left Stick]   →  Throttle
            [Right Stick]  →  Turn
